# REACT-NATIVE-SIGAC

Visualizador/Gerenciador de informações do SIGAC desenvolvido em React Native. Este projeto foi construído para oferecer uma experiência mobile fluida, moderna e responsiva para o gerenciamento de dados do sistema.

---

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias e conceitos:

* **React Native** — Framework para construção de aplicativos nativos usando React.
* **JavaScript / TypeScript** — Linguagem base do projeto.
* **React Navigation** — Gerenciamento de rotas e navegação entre telas.
* **Estilização** — Componentes customizados focados em uma interface limpa e moderna (Suporta Dark Mode / Glassmorphism se aplicável).

---

## 📦 Pré-requisitos

Antes de iniciar, você vai precisar ter instalado em sua máquina:

1.  **Node.js** (Versão LTS recomendada)
2.  **Gerenciador de pacotes** (`npm` ou `yarn`)
3.  **Ambiente React Native configurado** (Android Studio para emulador Android e/ou Xcode para emulador iOS). Caso tenha dúvidas, siga o [Guia oficial do React Native](https://reactnative.dev/docs/environment-setup).

---

## 🛠️ Como Executar o Projeto

Siga os passos abaixo para rodar a aplicação localmente:

### 1. Clonar o Repositório
```bash
git clone [https://github.com/gabrielfranca42/REACT-NATIVE-SIGAC.git](https://github.com/gabrielfranca42/REACT-NATIVE-SIGAC.git)
cd REACT-NATIVE-SIGAC

## Instalar Dependências

npm install
# ou, se usar yarn:

## Iniciar o Metro Bundler

npm start
# ou

## Rodar no Emulador ou Dispositivo Físico

# ou
